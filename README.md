
# 🛍️ Retail Chain Executive Dashboard

A modern **Retail Chain Executive Dashboard** built with **Python, PyQt5, Matplotlib, Pandas, and NumPy**. The application provides an interactive Business Intelligence (BI) experience with real-time cross-filtering, drill-down analysis, KPI tracking, and sales performance visualization similar to Microsoft Power BI.

## ✨ Features

- 📊 Interactive executive dashboard
- 📈 Sales performance analysis by Region, Store, Category, and Product
- 🔍 Cross-filtering and drill-down functionality
- 📅 Year-over-Year (YoY) sales comparison
- 🎯 Target vs Actual performance tracking
- 📉 Monthly sales trend visualization
- 📦 Product and category performance analysis
- 📍 Breadcrumb navigation and filter management
- 💻 Modern desktop GUI using PyQt5

## 🛠️ Technologies Used

- Python
- PyQt5
- Matplotlib
